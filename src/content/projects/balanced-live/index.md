BalancedLive is a cloud-based accounting platform that combines traditional accounting workflows with AI-powered automation. Built for small to mid-sized businesses, it supports multiple user roles including business owners, bookkeepers, and accountants.

## Key Features

- Multi-tenant architecture with PostgreSQL schema-based isolation
- Plaid banking integration for automatic transaction sync
- AI-powered transaction categorization using AWS Bedrock
- Full accounting suite: chart of accounts, journal entries, general ledger
- Accounts payable and receivable management
- Financial reporting (balance sheets, income statements)
- Built with React 19, TanStack Start (SSR), Django, and PostgreSQL

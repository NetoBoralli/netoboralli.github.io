Neo Health is a personal AI health platform that combines clinical data, lifestyle habits, and wearable insights to deliver personalized preventive care guidance available 24/7.

## Key Features

- AI-powered health advisor for personalized guidance
- Baseline nutrition assessment and analysis
- Personalized weekly meal plans with grocery lists
- Progress tracking dashboard for meals, habits, and results
- Built with React Native (mobile), Astro.js (marketing site), AWS Lambda, and AWS Bedrock
